# Work Day Scheduler Starter Code
Weekly Day Planner

Built with 
  
  1) HTML
  2) CSS
  3) Javascript
  4) jQuery
  5) Bootstrap

The purpose for this week's challange was to build a week planner in which the user can log in different tasks, delete and edit them with no complications.
